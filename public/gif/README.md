# GIF/Image Folder

Place your GIFs or images in this folder for the Valentine page "No" button feature.

## Instructions

1. Add your GIFs/images to this `/public/gif/` folder
2. Name them as: `gif1.gif`, `gif2.gif`, `gif3.gif`, ... up to `gif20.gif`
   - OR use image formats: `gif1.jpg`, `gif1.png`, `gif1.jpeg`, etc.
   - The code supports: `.gif`, `.jpg`, `.jpeg`, `.png`, `.webp`
3. The component will randomly pick and display one GIF/image each time the "No" button is clicked
4. Images will accumulate on screen scattered randomly until the "No" button disappears (after 20 clicks)

## Examples

- `/public/gif/gif1.gif`
- `/public/gif/gif2.jpg`
- `/public/gif/gif3.png`
- `/public/gif/gif4.jpeg`
- ... up to `gif20` (any supported format)

## File Formats Supported

- GIF (`.gif`)
- JPEG (`.jpg`, `.jpeg`)
- PNG (`.png`)
- WebP (`.webp`)

The component will randomly try different extensions, so you can mix formats!
